# Real-Time Multilingual Query Handler

This project is the implementation and working of a real time MultiLanguage Query Handler. It uses python's fastapi for the backend and the frontend is built using next.js, this is an AI pipeline that detects incoming query language, translates it to English, and routes it for response.

This Project Uses:
>AI for detecting languages

>AI for translation between languages

>AI-supported intelligent responses

Outcome
A minimal multilingual support interface that takes queries in any language, translates them into English in real-time, and optionally generates a predefined response.

## Installation

```sh
pip install -r requirements.txt
```
